      "suggested_new_stop": 数字或null
    }
  ]
}

重要说明：
- 只输出 JSON，不要 markdown 代码块
- action=REDUCE 时 shares_to_sell 为必填（= 总股数 × suggested_reduce_pct / 100，向下取整）
- action=HOLD 或 EXIT 时 shares_to_sell 必须为 null
- 若无新交易，new_trade 设为字符串 "NO NEW TRADE"
- 若无加仓动作，add_on_trades 设为空数组 []
- 若无加仓否决，add_on_vetoes 设为空数组 []
- position_actions 必须包含所有持仓——即使操作为 HOLD
